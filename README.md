ğŸ“– README: How to Run the Application
This project follows a professional MVC architecture to ensure the code is modular and easy to manage.

1. Prerequisites
Node.js (v18 or higher)

npm or yarn

Supabase Account (to host the SQL database)

2. Installation
Clone the Repository:

Bash
git clone <your-repository-url>
cd backend-assignment
Install Dependencies:

Bash
npm install
3. Environment Setup
Create a .env file in the root directory and add the following keys:

Code snippet
```
PORT=3000
SUPABASE_URL=your_supabase_project_url
SUPABASE_ANON_KEY=your_supabase_anon_key
GEO_REVERSE_API_KEY=your_geo_key
```
4. Running the Project
Development Mode: Runs with auto-reload using nodemon.

Bash
npm run dev
Production Mode:

Bash
npm start
